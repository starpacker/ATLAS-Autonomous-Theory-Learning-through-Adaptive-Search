# Project Phasing: AI-Driven Physics Discovery

---

## 总览

```
Phase 1: 重现经典 — 在量子实验上发现 F=ma 级别的公式
         目标: E=hf, Δλ=C(1-cosθ), Bragg条件
         价值: 证明系统能从量子实验数据中提取定律
         难度: ★★☆☆☆ (与 AI-Newton 发现 F=ma 对等)

Phase 2: 通用常数 — 跨实验发现 h
         目标: 从多个独立实验的公式中识别出同一个基本常数
         价值: 首次自动化发现普朗克常数
         难度: ★★★☆☆ (AI-Newton 发现了 G, 但我们的常数更隐蔽)

Phase 3: 框架发现 — 发现概率性、离散性、新数学结构
         目标: 开放性探索, 尝试让系统发现经典框架无法描述的现象
         价值: 如果成功, 是该领域的根本性突破
         难度: ★★★★★ (无先例)
```

每个 Phase 都是独立可发表的完整工作。Phase 3 失败不影响 Phase 1+2 的价值。

## Phase 1: 单实验定律发现

### 一句话目标

**在匿名化的量子实验数据上，用零物理先验的符号回归系统，自主发现各实验的核心公式。**

### 为什么这是可行的

| 类比 | AI-Newton 的成就 | 我们的目标 |
|------|-----------------|-----------|
| 数据类型 | 球+弹簧的轨迹 | 光电效应的电流-频率数据 |
| 目标公式 | F = ma | E_k = hf - W |
| 数学结构 | 线性关系 (二阶导数) | 线性关系 (含截止) |
| 变量数 | 3 (F, m, a) | 2-4 |
| SR难度 | 已证明可行 | 相当或略高 |

光电效应的 E_k = hf - W 本质上是一个带截止的线性关系——从SR角度看，和 F=ma 难度相当。康普顿散射的 Δλ = C(1-cosθ) 也是简单的三角函数关系。这些都在 PySR 的能力范围内。

### 具体包含的实验与目标公式

```
实验 1: 光电效应
  输入: 光源参数(频率), 光强, 材料选择, 外加电压
  输出: 电流表读数
  系统应发现: meter = C₁ · max(knob_0 - C₂, 0) · knob_1
              (即: 光电流 ∝ 光强, 存在截止频率, 截止以上线性)
  SR难度: 低 — 分段线性

实验 2: 康普顿散射
  输入: 入射源参数, 探测角
  输出: 散射信号探测器读数
  系统应发现: detector_0 - knob_0 = C₃ · (1 - cos(knob_1))
  SR难度: 低 — 余弦关系

实验 3: 电子衍射
  输入: 加速参数, 晶体选择, 探测角
  输出: 衍射强度
  系统应发现: 强度峰满足 C₄ · sin(knob_2) = n · C₅ / sqrt(knob_0)
  SR难度: 中 — 含平方根和整数条件

实验 4: 氢原子光谱
  输入: 观测频率范围
  输出: 离散发射线频率列表
  系统应发现: f_n = C₆ · (1/m² - 1/n²), n,m ∈ 整数
  SR难度: 中 — 需要发现整数结构

实验 5: 黑体辐射
  输入: 温度, 频率
  输出: 辐射强度
  系统应发现: I = C₇·f³ / (exp(C₈·f/T) - 1)
  SR难度: 高 — 含指数和除法的复合函数
```

### 方法选择（基于文献综述）

```
核心引擎: PySR
  理由: 文献中最强的开源SR, Pareto前沿, BFGS常数优化
        已在物理发现中验证 (暗物质晕浓度公式)

辅助策略 1: AI Feynman 式的递归分解
  对每个实验, 先训练NN作为函数预言机
  用NN检测对称性 (平移/缩放不变性)
  用NN检测可分离性 (加法/乘法可分)
  分解后对子问题做SR

辅助策略 2: AI-Newton 式的受控变量分析
  固定其他变量, 扫描一个变量
  检测: 线性? 二次? 三角? 指数? 阈值?
  用检测结果缩小SR搜索空间

概念提取: AI-Newton 方法
  从发现的公式中提取常数 → intrinsic / universal 分类
  从公式结构中提取依赖关系 → 概念图谱
```

### 环境设计（遵守反作弊规范）

```
所有输入: knob_0..knob_n ∈ [0, 1] (归一化)
所有输出: detector_0..detector_m (原始探测器读数)
无衍生量, 无布尔开关, 无物理语义
参数顺序每次运行随机化
```

### 成功标准（通用指标，不引用已知答案）

```
1. 系统发现的公式在留出测试集上 R² > 0.99
2. 公式的描述长度 (MDL) 在 Pareto 前沿上
3. 替代物理测试通过: 改变环境常数后, 系统发现不同的公式常数值
4. 事后人类检查: 发现的公式是否等价于已知物理定律
```

### 交付物

```
1. 5 个清洁环境模拟器的代码
2. 匿名化包装层
3. PySR + AI Feynman 式分解的集成系统
4. 5 个实验各自发现的 Pareto 最优公式集
5. 替代物理测试结果
6. 技术报告 / 论文草稿
```

### 估计周期

```
环境实现: 3-4 周
SR系统集成: 2-3 周
实验运行+调参: 3-4 周
验证+报告: 2 周
总计: ~3 个月
```

## Phase 2: 跨实验通用常数发现

### 一句话目标

**从 Phase 1 中各实验独立发现的公式中，自动识别出跨实验共享的基本常数（即 h），并用统一常数重写所有公式。**

### 为什么这是可行的但比 Phase 1 难

AI-Newton 成功发现了万有引力常数 G。但我们面临的挑战更大：

```
AI-Newton 发现 G:
  实验1: F = G · m₁m₂/r²    → G 直接出现
  实验2: F = G · m₁m₂/r²    → 同样形式, 同一个G
  → G 在所有实验中以相同形式出现, 数值聚类即可

我们发现 h:
  光电效应:    C₁ = h            → 直接
  康普顿散射:  C₃ = h/(m_e·c)    → h 与其他未知量复合
  电子衍射:    C₅ = h/√(2m_e·e)  → 更复杂的复合
  黑体辐射:    C₈ = h/k_B        → 与另一个未知常数复合
  → h 以不同的组合形式出现, 简单聚类行不通
```

### 核心方法

**方法 1: PSLQ 整数关系搜索（文献综述中识别的最干净工具）**

```
输入: Phase 1 中各实验发现的所有常数值
  {C₁, C₂, C₃, C₄, C₅, C₆, C₇, C₈, ...}

Step 1: 取对数 → 将乘法关系转化为加法关系
  {log|C₁|, log|C₂|, log|C₃|, ...}

Step 2: PSLQ 搜索整数关系
  寻找整数向量 (n₁, n₂, ...) 使得
  n₁·log|C₁| + n₂·log|C₃| + n₃·log|C₅| + ... ≈ 0
  即: C₁^n₁ · C₃^n₂ · C₅^n₃ · ... ≈ 1

Step 3: 从整数关系中提取"基本常数"
  如果发现 C₁ / (C₃ · C_extra) ≈ 1
  → C₁ 和 C₃ 共享一个因子
  → 提取最小基: {UC₁, UC₂, UC₃, ...} 使得所有 Cᵢ 都能表示为基的幂积

Step 4: 用统一常数重写所有 Phase 1 的公式
```

**方法 2: AI-Newton 式合情推理**

```
如果光电效应公式含常数 C₁
且康普顿公式含常数 C₃
且 C₁/C₃ 可被另一个实验中的常数解释
→ 合情推断: C₁ 和 C₃ 共享基本因子
→ 定义新概念: UC₁ = C₁, 并检验 C₃ = UC₁/(...其他概念...)
```

**方法 3: 共享参数的联合 SR**

```
灵感来源: MASS 的共享线性层 + PySR/GNN 的组件分解

对多个实验的数据同时做 SR
但约束: 不同实验的公式可以共享常数
搜索: 最小化 (总拟合误差 + 常数个数惩罚)
如果共享一个常数能在不损失精度的情况下减少总复杂度 → 接受共享
```

### 前置依赖

Phase 1 必须成功发现至少 3 个实验的公式，且这些公式中包含的常数足够精确（相对误差 < 1%），才能进入 Phase 2。

### 成功标准

```
1. 系统识别出的"基本常数集"大小 ≤ 5
2. 所有 Phase 1 的公式都能用基本常数集重写
3. 重写后的公式精度不下降 (R² 不变)
4. 替代物理测试: h→2h 后, 系统发现的 UC₁ 也变为 2 倍
5. 事后检查: UC₁ 是否等于 h (在归一化反变换后)
```

### 交付物

```
1. 常数关系发现算法实现 (PSLQ + 合情推理)
2. 跨实验统一的公式集
3. 发现的基本常数表
4. 替代物理测试结果
5. 技术报告 / 论文
```

### 估计周期

```
PSLQ实现+集成: 2 周
合情推理模块: 3-4 周
联合SR实验: 3-4 周
验证+报告: 2 周
总计: ~3 个月 (与 Phase 1 部分可并行)
```

## Phase 3: 框架发现 — 开放性探索

### 一句话目标

**探索让系统自主发现"现有数学框架不够用"的机制，以及可能的扩展方向。**

### 为什么这是未解决的

文献综述的核心结论：

> 没有任何现有方法发现过数学框架。所有方法都在预设框架内搜索。

但这正是这个 Phase 的价值——如果能取得任何进展，都是该领域的根本性突破。

### 我们目前不知道的

```
- 如何让系统"意识到"实数域不够用
- 如何让系统"发明"复数（或其他新数学结构）
- 如何让系统发现"确定性描述不够用，需要概率性描述"
- 如何让系统发现 Born 规则 P=|ψ|²
- 这些是否可能在没有人类引导的情况下发生
```

### 我们知道的起点

尽管不知道最终答案，但有几个有希望的方向可以尝试：

**方向 A: 让系统发现"SR 失败"本身是有意义的**

```
Phase 1 中, 如果某些实验(如双缝干涉的低强度区域)
SR 无法找到 R² > 0.99 的确定性公式
→ 这不是系统的失败, 而是一个发现:
   "这个实验的输出不能被确定性公式描述"

具体做法:
  1. 对所有实验运行标准SR
  2. 记录SR失败或R²显著偏低的区域/实验
  3. 分析这些失败案例的共同特征:
     - 相同输入多次实验, 输出是否有随机性?
     - 输出的方差是否取决于输入?
     - 输出的统计分布是否有规律?
  4. 如果发现输出是随机的 → 切换到"拟合分布"而非"拟合值"
  5. 对分布的参数做SR → 这可能导向概率性定律的发现

关键: 系统不是被告知"要找概率性定律"
     而是从"确定性SR失败"中自主得出这个结论
```

**方向 B: SciNet 式的表示发现——瓶颈维度搜索**

```
灵感来源: SciNet 通过信息瓶颈发现了日心说

对每个实验训练 SciNet 风格的编码器-解码器:
  瓶颈维度 = 1 → 如果预测好 → 1维描述够用
  瓶颈维度 = 2 → 如果预测好 → 2维描述够用
  ...

关键问题: 对于量子实验
  - 用实数瓶颈, 需要 N 维才能好好预测
  - 用复数瓶颈, 只需要 N/2 维?
  → 如果系统发现"复数瓶颈更高效"→ 自主发现了复数结构的必要性

但这需要同时搜索瓶颈的数据类型(实/复)
这是否构成作弊? → 不清楚, 需要仔细讨论
```

**方向 C: 从"拟合失败的模式"中学习**

```
灵感来源: MASS 中的理论转换 (Hamiltonian → Lagrangian)

假设系统在 Phase 1 中发现:
  - 高强度双缝: 输出是平滑的周期性分布 → SR成功, 找到波动公式
  - 低强度双缝: 输出是稀疏的离散尖峰 → SR失败
  - 光电效应: 输出是确定性的线性关系 → SR成功

那么系统面前有一个模式:
  "同一实体(entity_A)在某些条件下可被确定性公式描述,
   在另一些条件下不能"

如何从这个模式中推进?
  → 方向A的"拟合分布"策略
  → 然后比较: 低强度双缝的统计分布是否 = 高强度双缝的确定性分布?
  → 如果是 → 系统自主发现了"概率分布 = 波动强度"(即 Born 规则的雏形)
```

**方向 D: 维度扩展搜索（来自 v3 critique）**

```
如果两个模型(一个描述波行为, 一个描述粒子行为)
对同一实体的不同实验分别有效但互相不兼容

搜索: 是否存在一个扩展模型
  在"条件Z=z₁"下退化为波模型
  在"条件Z=z₂"下退化为粒子模型?

Z 可能是: 粒子数、检测方式、统计聚合方式...
这不需要预设答案, 只是一个通用的"模型统一"搜索
```

### Phase 3 的特殊性质

```
与 Phase 1/2 的根本区别:

Phase 1: 目标明确, 方法已验证, 成功概率高
Phase 2: 目标明确, 方法需适配, 成功概率中等
Phase 3: 目标开放, 方法需发明, 成功概率未知

Phase 3 的价值不在于"一定成功"
而在于:
  - 清晰地定义"框架发现"这个问题
  - 排除一些不可行的方向
  - 可能发现一些 partial results
  - 即使完全失败, 对"框架发现为什么难"的分析也有学术价值
```

### 成功标准（开放性的）

```
Level 0 (有价值的失败):
  清晰地分析了为什么框架发现不可行
  排除了若干看似可能但实际不行的方法
  → 可作为 position paper / negative result 发表

Level 1 (初步进展):
  系统自主检测到"某些实验数据不能被确定性公式描述"
  系统切换到概率分布拟合, 并发现分布有规律
  → 可作为 workshop paper 发表

Level 2 (有意义的发现):
  系统发现"粒子到达的统计分布 = 波动强度分布"
  即 P(x) ∝ I(x), 其中 I 是 Phase 1 发现的波动公式
  → 这是 Born 规则的雏形, 高价值论文

Level 3 (突破):
  系统自主发现需要复数或新数学结构来统一描述
  → 如果实现, 是该领域的里程碑
```

### 交付物

```
1. 各方向的探索代码和实验记录
2. 成功/失败的详细分析
3. 对"框架发现"问题的形式化定义
4. 技术报告 / 论文 (根据达到的 Level 选择投稿级别)
```

### 估计周期

```
开放性探索, 不设硬性截止
建议: 在 Phase 1+2 基本完成后, 用 4-6 个月探索
定期 (每月) checkpoint, 评估是否有进展值得深入
```

## 跨 Phase 的通用基础设施

以下组件在所有 Phase 中共享：

```
1. 环境层
   - 清洁接口 (knob/detector, 归一化, 无语义)
   - 替代物理支持 (可调 h, m_e, c 等常数)
   - 噪声控制 (可调噪声级别)

2. 反作弊框架
   - 替代物理测试套件
   - 随机数据测试
   - Anti-Cheating Checklist (每个PR必检)

3. 评估框架
   - 通用指标 (R², MDL, Pareto前沿)
   - 多种子一致性分析
   - 可视化工具

4. 实验记录
   - 完整的SR进化谱系
   - 所有中间结果存档
   - 可复现的随机种子管理
```
