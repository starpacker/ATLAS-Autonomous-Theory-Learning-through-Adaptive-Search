# Brainstorm v1: Framework Draft

## 核心突破：GPT (广义概率论) 将"框架发现"变为"参数搜索"

之前的问题：如何在无限的"可能框架"空间中搜索？
GPT 的回答：**这个空间不是无限的。** 它由少数几个"旋钮"参数化。

```
经典力学和量子力学不是"两种完全不同的东西"
它们是同一个参数空间中的两个点:

  Theory = GPT(K, N, composition, group, purification)

  经典:  GPT(K=N-1,   N, simplex_tensor, S_N,    no_purification)
  量子:  GPT(K=N²-1,  N, hilbert_tensor,  U(N),  purification)

  其中:
    N = 系统的可区分状态数 (distinguishability)
    K = 描述状态需要的实参数数 (state space dimension)
    composition = 复合系统的状态空间如何构成
    group = 允许的可逆变换群
    purification = 是否每个混态都有唯一纯化
```

**关键数字：对于 N=2 (最简单的非平凡系统):**

| 理论 | K | 状态空间几何 |
|------|---|------------|
| 经典 (coin) | 1 | 线段 [0,1] |
| 实量子 (rebit) | 2 | 圆盘 D² |
| 复量子 (qubit) | 3 | 球体 B³ (Bloch球) |
| 四元数 | 5 | 5维球 B⁵ |

→ K 的选择基本决定了理论的性质！
→ 搜索空间：对于 N=2，K ∈ {1, 2, 3, 5, ...} — 一个离散的、极小的搜索空间

## 新框架草案：GPT-Search

### 核心思想

```
Phase 1: 在经典框架(K=N-1)下做SR → 发现单实验公式
Phase 2: 跨实验常数统一 (PSLQ)
Phase 3: 当经典框架拟合失败时,
         搜索GPT参数空间,
         找到拟合数据的最优理论
         → 如果最优理论的参数 = 量子力学的参数,
           系统就"发现"了量子力学
```

### 为什么这比"文法搜索"更好

```
文法搜索 (之前的方案):
  - 搜索空间: ~百万级 (文法修改操作的组合)
  - 每步评估: 需要重新运行SR (慢)
  - 理论基础: ad hoc (修改操作是人为设计的)
  - 完备性: 不清楚是否覆盖了所有可能的理论

GPT搜索 (新方案):
  - 搜索空间: 极小 (几个离散/连续参数)
  - 每步评估: 在GPT框架下做拟合 (数学上有解析结构)
  - 理论基础: 量子信息基础 (Hardy, Chiribella 等20+年研究)
  - 完备性: GPT被证明覆盖了所有"合理的"操作性理论
```

### 但有一个巨大的问题

**GPT 框架本身是否构成"作弊"？**

```
GPT 是物理学家在2000年代发明的,
明确目的就是"找到量子力学在理论空间中的位置"。

把GPT编码到系统中 = 告诉系统:
  "物理理论由这些参数决定,
   你需要搜索的是这些参数的值"

这与把 |ψ|² 编码到搜索空间中有什么区别？

可能的辩护:
  GPT 不是特指量子力学 — 它同样包含经典物理
  GPT 的"旋钮"是信息论原则,不是特定物理理论
  类比: 符号回归的运算符集 {sin, cos, exp} 也是"先验",
        但被社区接受为"通用数学工具"

反驳:
  GPT 是从量子力学的反思中产生的
  一个不知道量子力学的文明不会想到GPT
  {sin, cos, exp} 是通用数学; GPT 不是通用数学
```

**这是一个需要正面解决的问题。暂时搁置，继续探索。**

## 替代思路：从数据中发现 GPT 参数，而非预设 GPT

如果直接用GPT框架是作弊，能否让系统**自己发现** GPT 的结构？

### 思路：信息瓶颈 + 操作性重构

```
Step 1: SciNet 式的信息瓶颈
  对每个实验, 训练编码器-解码器
  寻找最小瓶颈维度 K_min

Step 2: 对不同实验比较 K_min
  经典预期: K_min = N-1 (N个可区分状态)
  如果发现 K_min > N-1 → 经典框架不够!
  如果发现 K_min = N²-1 → 恰好是量子力学的维度

Step 3: 从 K_min 推断理论结构
  K=1 for N=2 → 经典
  K=3 for N=2 → 量子(复数)
  K=2 for N=2 → 量子(实数)

这个方法:
  ✓ 不预设GPT框架
  ✓ 从数据中测量关键参数(K)
  ✓ K的值直接告诉你需要什么类型的数学
  ✗ 但K的解释(K=N²-1意味着什么)仍需要理论框架
```
